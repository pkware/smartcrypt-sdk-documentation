Modern encryption methods that cover most application requirements.

Fields

Name Description
Ae2

Provided for compatibility with other ZIP applications, such as WinZIP. Should only be used if compatibility is a requirement, otherwise, prefer Aes256.
Aes256

The most preferred encryption method. This should be used by all new applications, unless requirements dictate otherwise.
None

No encryption should be performed.

A type alias for the algorithm that is recommended. This may change between software releases.
