Interface IArchiveFactory

Serves as a replacement for PKWARE.ArchiveAPI.Factory. Be sure to call Dispose() when done. Consider the lifespan of this factory to be the same as that of the archive you are working with. If working with multiple archives, it is recommended to use a new factory for each archive.

The newly configured set of options.
Remarks

This serves as an alternative to PKWARE.ArchiveAPI.Factory.CreateItemOptions and should be used instead. This variant returns options that are preconfigured with reasonable defaults for Smartcrypt. Most developers will not need to modify these defaults. These defaults are subject to change between releases of the SDK.
